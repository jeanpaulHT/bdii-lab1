## Usage:

### On command line:

```
git clone https://github.com/jeanpaulHT/bdii-lab1.git
cd source
g++ p1.cpp             # same for p2, p3 or p4
./a.out                # or a.exe for windows
```

### On Clion:

- Open CMakeLists.txt
- Comment out the line which corresponds to the test you want to run
- Run the project
